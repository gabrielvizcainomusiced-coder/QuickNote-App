# QuickNote

QuickNote is a simple note-taking app built with React.

It allows users to create, edit, and delete notes. Notes are displayed in a responsive grid layout and support long text content.

## Features

- Create notes with a title and content
- Edit existing notes
- Delete notes
- Responsive layout
- Footer that stays at the bottom of the page
- Notes grow naturally based on content length

## Tech Stack

- React
- Vite
- JavaScript
- CSS
